    Código Título Descripción Actor Precondición Postcondición Flujo Alternativo
    CU01 Visualizar Lista de Facturas El usuario puede visualizar la lista de facturas en un rango de fechas. Usuario Usuario autenticado y con permisos para ver facturas. Se muestra la lista de facturas filtradas por fecha. No hay facturas en el rango de fechas seleccionado.
