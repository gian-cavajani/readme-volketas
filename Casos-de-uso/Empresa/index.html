    CU01 - Ver Lista de Empresas¶

    CU ID Caso de Uso Descripción Actor Precondición Resultado Esperado Flujo Alternativo
    CU01 Ver Lista de Empresas El usuario puede ver una lista de empresas filtradas por la letra seleccionada. Usuario El usuario ha iniciado sesión. La lista de empresas que comienzan con la letra seleccionada se muestra correctamente. No hay empresas que comiencen con la letra seleccionada.
    1. El sistema muestra un mensaje indicando que no hay empresas disponibles para esa letra.
    2. Se ofrece al usuario la opción de seleccionar otra letra.

    CU02 - Buscar Empresa por Nombre¶

    CU ID Caso de Uso Descripción Actor Precondición Resultado Esperado Flujo Alternativo
    CU02 Buscar Empresa por Nombre El usuario puede buscar empresas ingresando un nombre en la barra de búsqueda. Usuario El usuario ha iniciado sesión. Los resultados de la búsqueda se muestran correctamente basados en el nombre ingresado. No hay empresas que coincidan con el término de búsqueda.
    1. El sistema muestra un mensaje indicando que no hay resultados que coincidan con el término de búsqueda.
    2. Se ofrece al usuario la opción de modificar el término de búsqueda o reiniciarlo.

    CU03 - Crear Nueva Empresa¶

    CU ID Caso de Uso Descripción Actor Precondición Resultado Esperado Flujo Alternativo
    CU03 Crear Nueva Empresa El usuario puede crear una nueva empresa ingresando los datos requeridos. Usuario El usuario tiene permisos para crear empresas. La nueva empresa se crea y se guarda correctamente en el sistema.
    Navega a la empresa creada en caso de no estar en pantalla de crear pedido.    		 Error al crear la empresa.
    1. Si ocurre un error en la validación de los datos, el sistema muestra un mensaje de error específico.
    2. El usuario corrige los datos y vuelve a intentar crear la empresa.
    3. Si el error persiste, el sistema muestra un mensaje de error general y registra el error en el log del sistema.
